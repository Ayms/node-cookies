node-cookies
===

Simple implementation of cookies

## Install :

    npm install node-cookies

or

    git clone http://github.com/Ayms/node-cookies.git
    cd node-cookies
    npm link .

## Use :
See :
	[Ayms/node-dom](https://github.com/Ayms/node-dom)
